# 🌦️ Projet Météo - Java Spring Boot + API Open-Meteo

Ce projet est une petite application Java utilisant **Spring Boot** qui :

- Fait des appels **GET** et **POST** vers l’API météo gratuite [https://open-meteo.com](https://open-meteo.com)
- Récupère les données météo au format **JSON**
- Affiche le résultat dans une petite **page web HTML** avec un bouton pour lancer requête.

---

## 🚀 Objectifs

- Créer une API REST avec Spring Boot
- Consommer une API externe (Open-Meteo)
- Afficher les données météo dans une page HTML simple

---

## 🧰 Stack utilisée

- Java 21
- Spring Boot
- Maven
- API : [Open-Meteo](https://open-meteo.com/)
- HTML / JavaScript (frontend simple)

---

## 🔧 Installation du projet

### 1. Initialisation du projet Spring Boot

Tu peux utiliser [https://start.spring.io](https://start.spring.io) avec cette configuration minimale :

- **Project**: Maven
- **Language**: Java
- **Group**: `com.example`
- **Artifact**: `meteoapp`
- **Name**: `meteoapp`
- **Dependencies**:
    - `Spring Web`
    - `Gson google`
    - `DevTools`
